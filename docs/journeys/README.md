## Download / PDF

1. Open the folder `docs/journeys/` in Explorer or download the repo zip / this folder.
2. For PDF: open `LVLUP-JOURNEY-GUIDE.md` in any Markdown viewer (or VS Code preview) → **Print → Save as PDF**. (The print-styled `.html` is a generated build product and is not tracked; regenerate it locally if you need it.)
