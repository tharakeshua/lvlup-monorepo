# LeaderboardRow

A single ranked leaderboard entry composing `Avatar`. Shows rank (top-3
emphasized), the learner's avatar and name, and their XP. Set `me` to highlight
the viewer's row.

```jsx
<LeaderboardRow rank={1} initials="SK" name="Subhang K." xp={4820} />
<LeaderboardRow rank={4} initials="ME" name="You" xp={3110} me />
```
