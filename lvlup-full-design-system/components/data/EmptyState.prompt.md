# EmptyState

No-data placeholder with a lucide `icon`, `title`, `body`, and an optional
`action` node (usually a Button). Defaults `icon` to `inbox`.

```jsx
<EmptyState
  icon="folder-open"
  title="No spaces yet"
  body="Create your first learning space to get started."
  action={<Button variant="primary">New space</Button>}
/>
```
