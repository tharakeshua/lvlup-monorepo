# SpaceCard

A subject-space tile showing the space title, an optional description, the
story-point count, and mastery progress. Set `spark` to highlight a
featured/active space banner.

```jsx
<SpaceCard
  title="Data Structures & Algorithms"
  description="Master arrays, trees, graphs, and dynamic programming."
  points={48}
  progress={62}
  spark
/>
```
