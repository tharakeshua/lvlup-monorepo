# V13: Marketing Website & Landing Pages - Cycle 3 Plan

## Overview

Create a standalone Astro static marketing site at `website/` for Auto LevelUp,
sharing design tokens from `packages/tailwind-config/`.

## Architecture

- **Framework**: Astro 5.x (static site generation)
- **Styling**: Tailwind CSS with shared `@levelup/tailwind-config` tokens
- **Integrations**: `@astrojs/sitemap`, `@astrojs/tailwind`
- **Build Output**: Static HTML/CSS/JS in `website/dist/`

## Pages & Components

### Pages

1. **Homepage** (`/`) - Hero, Features, How It Works, Testimonials, FAQ, CTA
2. **Pricing** (`/pricing`) - 4-tier pricing grid (Free/Basic/Pro/Enterprise),
   feature comparison table
3. **Contact** (`/contact`) - Contact form with validation, contact info, demo
   request

### Components

| Component            | Description                                                 |
| -------------------- | ----------------------------------------------------------- |
| `Layout.astro`       | Base HTML layout with SEO meta, Open Graph, JSON-LD         |
| `Nav.astro`          | Floating nav with scroll progress bar, mobile menu          |
| `Hero.astro`         | Gradient animation hero with product preview mock           |
| `Features.astro`     | 4-role feature cards (Schools, Teachers, Students, Parents) |
| `HowItWorks.astro`   | 4-step timeline walkthrough                                 |
| `Testimonials.astro` | Horizontal scroll carousel with nav buttons                 |
| `FAQ.astro`          | Accordion FAQ with 8 questions                              |
| `CTA.astro`          | Gradient CTA banner with dual buttons                       |
| `Footer.astro`       | 4-column link footer with social icons                      |

## SEO Deliverables

- Meta tags (title, description, canonical) on all pages
- Open Graph tags (og:title, og:description, og:image, og:type)
- Twitter Card meta tags
- JSON-LD structured data (SoftwareApplication, ContactPage)
- `sitemap-index.xml` auto-generated by `@astrojs/sitemap`
- `robots.txt` with sitemap reference

## UX Deliverables

- Gradient animation on hero background
- Scroll-triggered reveal animations (IntersectionObserver)
- Floating nav with backdrop blur on scroll
- Scroll progress indicator bar
- Testimonial carousel with prev/next buttons
- FAQ accordion with smooth transitions
- Mobile hamburger menu with slide-in
- Responsive design (mobile-first)
- `prefers-reduced-motion` support
