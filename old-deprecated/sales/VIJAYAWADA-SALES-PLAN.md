## 1. PRODUCT SCOPE SUMMARY

### What is Auto LevelUp?

An **all-in-one AI-powered educational platform** with 5 dedicated portals:

| Portal             | For                  | Key Value                                           |
| ------------------ | -------------------- | --------------------------------------------------- |
| **Super Admin**    | Platform owner       | Multi-school management                             |
| **School Admin**   | Principal/Management | School-wide analytics, teacher & student management |
| **Teacher Portal** | Teachers             | AI AutoGrade, assignment creation, digital testing  |
| **Student Portal** | Students             | Gamified learning, XP system, skill progression     |
| **Parent Portal**  | Parents              | Real-time progress tracking, teacher communication  |

### Key Differentiators

1. **AI AutoGrade** — Grades assignments instantly using Google Gemini AI
   (essays, code, MCQs, audio)
2. **Gamification** — XP system with Silver/Gold/Platinum/Diamond skill tiers
   keeps students engaged
3. **15 Question Types** — MCQ, code, paragraph, numerical, matching,
   fill-blanks, audio, image evaluation
4. **Multi-Tenant** — Each school gets isolated, branded space
5. **Real-Time Analytics** — Dashboard for teachers, admins, and parents
6. **Digital Testing** — Complete online exam platform with timed tests,
   adaptive difficulty
7. **Parent Engagement** — Dedicated portal for parent involvement
8. **LIVE & DEPLOYED** — Not a prototype. Fully functional at
   https://lvlup-ff6fa-website.web.app

### Live URLs

- Teacher Portal: https://lvlup-ff6fa-teacher.web.app

## 7. OBJECTION HANDLING PLAYBOOK

| Objection                        | Response                                                                                                                                             |
| -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| "We use Google Classroom"        | "Great! We complement it. Google Classroom doesn't have AI grading, gamification, or parent dashboards. Many schools use both."                      |
| "Too expensive"                  | "Start free with 30 students. When you see teachers saving 10+ hours/week, the ₹5K/mo pays for itself. That's less than one teacher's daily salary." |
| "Our teachers aren't tech-savvy" | "Our interface is simpler than WhatsApp. We provide 2 free in-person training sessions and a dedicated WhatsApp support group."                      |
| "Students will get distracted"   | "Our gamification is designed for learning, not entertainment. XP is earned only through completing assignments and scoring well."                   |
| "What about internet issues?"    | "Works on any device with basic internet. Assignments are lightweight — no heavy downloads needed."                                                  |
| "Need management approval"       | "Absolutely. Can I present to your management? I'll prepare a customized proposal for [School Name]."                                                |
| "We'll think about it"           | "Of course. In the meantime, can I set up a free pilot for just one class? Zero commitment, and you'll see real results in a week."                  |
| "Is our data safe?"              | "100%. Each school's data is completely isolated. We're FERPA and COPPA compliant. Data is encrypted and hosted on Google Cloud."                    |
| "What subjects do you support?"  | "Any subject. We come pre-loaded with Math, Science, and English content. You can add any subject and your teachers can create content easily."      |
